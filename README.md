##	practice02 动态规划

题目：有 n 个货物，一名工人同时可以搬运 2 个货物，此时会消耗体力 $(x - y)^4$ 问 m 次搬运消耗的体力值最少为多少？

**输入格式：**
<br>第一行为 n 和 m
第二行为 n 个数字

**输出格式：**
<br>消耗的最少体力值

**输入样例：**
<br>9  4
<br>1 3 7 9 10 12 17 20 22

**输出样例：**
<br>64
